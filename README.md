# LD-V4300D-Service-Adaptor
A service adaptor for the Pioneer LD-V4300D LaserDisc player

## Synopsis

This board is designed to assist with calibration of the Pioneer LD-V4300D LaserDisc player.  The board routes the various test signals to oscilloscope friendly terminals and also includes the passive components required for the service manual measurements.

## Motivation

Calibrating LD-V4300D players.

## Author

The LD-V4300D service adaptor board was designed by Simon Inns.
